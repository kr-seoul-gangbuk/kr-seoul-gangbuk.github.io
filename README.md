# 강북구 · 서울특별시

> 강북구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-gangbuk.github.io/**

강북구는 북한산 백운대와 인수봉이 병풍처럼 감싸 안은 서울의 대표적인 '숲세권' 자치구로, 도심 속에서 산과 계곡을 곧바로 누릴 수 있는 드문 생활권을 자랑합니다. 우이동 계곡과 솔밭근린공원의 소나무 군락, 빨래골로 불리던 수유동의 맑은 물줄기가 이어져 사계절 등산객과 나들이객의 발길이 끊이지 않습니다. 무엇보다 국립4·19민주묘지와 수유리 순국선열묘역, 근현대사기념관이 모여 있어 대한민국 민주주의와 독립운동의 정신이 살아 숨 쉬는 '역사의 성지'로 불립니다. 미아사거리와 수유역 일대의 활기찬 상권, 북서울꿈의숲의 넓은 녹지, 그리고 우이신설선이 실어 나르는 편리한 이동성이 어우러져 자연과 역사, 도시 생활이 균형을 이룬 동네입니다.
